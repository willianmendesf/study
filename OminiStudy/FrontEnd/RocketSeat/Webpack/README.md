# Mini Dev repository

Ambiente de desenvolvimento minimal para desenvolvimento de aplicações rápidas
e componentizadas. 

## Tecnologias:
- webpack
- babel 
- ES6
- Node

### Modo de Uso: 

```
Development:  [ yarn dev ]
```
```
Build:  [ yarn build ]
```
```
Babel: yarn babel
```
```
Webpack: yarn webpack
```
